            # Vuepress 部署到 github page

            # 使用vuepress 有太多坑要注意,故在此做筆記記錄

            由於github page是放置靜態資源,須先將專案文件打包成靜態資源

            步驟：

            1. 先確認config.js和要設定下面一項：
            // config.js 目錄
base: '/myNote/',
// dest: 'dist', // 補充：若設置此項目,dist文件會和docs目錄同一層
            1
            2
            3
            1. 在該專案根目錄下創建deploy.sh檔案,
              deploy.sh檔案內容設定如下：
















             



            #!/usr/bin/env sh

# 确保脚本抛出遇到的错误
set -e

# 生成静态文件
yarn docs:build

# 进入生成的文件夹
cd docs/.vuepress/dist

git init
git add -A
git commit -m 'deploy'

# 如果发布到 https://tiffany8053.github.io  填写你刚刚创建的仓库地址
git push -f https://github.com/tiffany8053/專案名稱.git master

cd -

            注意:

            記得修改上述第17行 https://github.com/github使用者帳號/專案名稱.git

            然後package.json要設定"deploy": "bash deploy.sh"




             


            "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs",
    "deploy": "bash deploy.sh"
  },

            提示

            這是為了之後直接輸入指令yarn deploy 就會自動執行deploy.sh檔案

            1. 將文件打包
            yarn docs:build
            1

            會在docs/.vuepress資料夾下多一個 dist文件夾

            1. 到github創建repository

            注意：

            Repository 一定要公開 public 才能使用github page

            1. 輸入下列指令執行deploy.sh檔案
            yarn deploy
            1
            1. 在到GitHub -> setting -> GitHub Pages none 改為 master,並save保存
